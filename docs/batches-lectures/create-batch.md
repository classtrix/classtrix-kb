# Create a Batch

## Purpose

Use this page to create a new batch for a course or session. A batch helps you group students together for lectures, attendance, exams, and communication.

## Steps

### Step 1

Open the Batches module and click the button to create a new batch.

### Step 2

Enter the batch name, course, branch, start date, and expected end date if your institute tracks it.

### Step 3

Set the capacity, class timing, and faculty member if these fields are available.

### Step 4

Save the batch and review the details before assigning students.

## Tips

- Use a clear naming format such as NEET Morning May 2026.
- Check the class timing before saving to avoid overlap with another batch.
- Create separate batches if morning and evening schedules differ.
