# Dashboard Overview

## Purpose

The dashboard gives you a quick view of the day’s work. Use it to monitor new enquiries, follow-ups, admissions, payments, and upcoming lectures without opening each module one by one.

## Steps

### Step 1

Sign in to Classtrix and wait for the dashboard to load fully.

### Step 2

Review the summary cards to check important numbers such as new leads, pending follow-ups, active batches, and today’s collections.

### Step 3

Look at the recent activity section to see the latest updates made by your team.

### Step 4

Use the menu or shortcuts on the dashboard to open the module where you need to work next.

## Tips

- Start each morning by checking pending follow-ups and today’s lectures.
- If a number looks incorrect, open the related module and confirm the latest entries.
- Use dashboard shortcuts to save time during busy admission periods.
