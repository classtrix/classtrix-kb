# Enter Exam Marks

## Purpose

Marks entry helps staff record student performance after an exam is completed. Use it to enter marks, mark absentees, and prepare result summaries.

## Steps

### Step 1

Open the Exams module and select the completed exam.

### Step 2

Go to the marks entry section for that exam.

### Step 3

Enter the marks for each student and mark absent students where required.

### Step 4

Save the marks and review the result summary before closing the page.

## Tips

- Keep answer sheets arranged by roll number to speed up entry.
- Save in between if you are entering marks for a large batch.
- Recheck very high or very low scores before publishing results.
