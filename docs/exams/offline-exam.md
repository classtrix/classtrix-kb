# Create an Offline Exam

## Purpose

Use this page to set up a pen-and-paper exam for a batch or course. It helps your team plan the exam date, subject, room details, and total marks in one place.

## Steps

### Step 1

Open the Exams module and click the option to create a new offline exam.

### Step 2

Choose the batch, exam name, subject, and exam date.

### Step 3

Enter the timing, total marks, pass marks, and room details if your institute uses them.

### Step 4

Save the exam and share the final schedule with the batch.

## Tips

- Use clear exam names such as Weekly Test 3 or Physics Mock Exam.
- Confirm the correct batch before publishing the exam.
- If the exam is rescheduled, update the record immediately so staff can see the latest plan.
