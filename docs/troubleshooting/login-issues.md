# Fix Login Issues

## Purpose

This page helps staff solve common login problems such as incorrect password, locked account, or browser issues. Use it before contacting support.

## Steps

### Step 1

Confirm that you are using the correct Classtrix login link and the right username or mobile number.

### Step 2

Re-enter the password carefully and check that Caps Lock is turned off.

### Step 3

If the password still does not work, use Forgot Password or ask your branch administrator to help reset access.

### Step 4

If the problem continues, try another browser or clear the browser cache, then contact support with a screenshot of the error.

## Tips

- Do not share your password with other staff members.
- Keep a screenshot of the error message if you need support.
- Mention your branch name and registered mobile number when reporting the issue.
