# Create a Lead

## Purpose

Use this page to add a new student enquiry into Classtrix. A lead can come from a phone call, walk-in visit, website form, social media campaign, or reference.

## Steps

### Step 1

Open the Leads module and click the button to add a new lead.

### Step 2

Enter the student name, mobile number, parent contact number if available, and city or area.

### Step 3

Select the course of interest, lead source, session, and counselor responsible for follow-up.

### Step 4

Add any useful notes such as preferred timing, budget discussion, or callback request, then save the lead.

## Tips

- Check the mobile number before saving to avoid duplicate enquiry records.
- Add clear notes if the student is interested in more than one course.
- Record the source properly so your team can review which campaigns bring the best enquiries.
